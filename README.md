# Список тестов
1. Заказ карты с валидными данными с добавлением 2 дней к дате по умолчанию
2. Заказ карты с валидными данными без изменения даты по умолчанию
3. Заказ карты с невалидной датой
4. Заказ карты с невалидной датой из прошедших дней
5. Заказ карты с названием города на английском языке
6. Заказ карты в город не являющийся административным центром
7. Заказ карты с невалидным именем
8. Заказ карты с невалидным нмоером телефонома
9. Заказ карты с непроставленным чекбоксом
